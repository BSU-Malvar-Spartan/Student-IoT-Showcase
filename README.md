## 🤝 Contributing

We welcome all BSU Malvar students to add their IoT projects! Please read [CONTRIBUTING.md](CONTRIBUTING.md) for full step-by-step instructions.

**Quick summary:**

1. **Fork** this repository.
2. **Edit** `README.md` — add a new row to the Project Showcase table with your project name, your name, a short description, and a link to your GitHub repo.
3. **Commit** your changes with a meaningful message (e.g., `Add [Your Project Name] by [Your Name]`).
4. **Open a Pull Request** back to this repository. A maintainer will review and merge it.

See [CONTRIBUTING.md](CONTRIBUTING.md) for more details and formatting guidelines.
